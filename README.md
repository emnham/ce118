# Tìm hiểu thiết kế bộ tăng tốc xử lý tín hiệu hình ảnh/video
**Môn học:** Thiết kế luận lý số (CE118)

## 1. Giới thiệu
Dự án tập trung nghiên cứu kiến trúc phần cứng giúp tăng tốc các thuật toán xử lý ảnh trên FPGA.

## 2. Các thuật toán triển khai
* Lọc nhiễu Median
* Phát hiện biên Sobel
* Chuyển đổi không gian màu RGB sang Gray

## 3. Công cụ sử dụng
* Ngôn ngữ: Verilog/SystemVerilog
* Phần mềm: Vivado, ModelSim
* Board mạch: DE2-115
